1.	Какой самый эффективный способ конкатенации строк?

Начиная с go 1.10 самый эффективный способ конкатенации строк - использование strings.Builder
В отличие от обычной конкатенации строк ("string" + "string"), где для каждой новой строки выделяется память (строки иммутабельны в go),
strings.Builder добавляет строки в один объект памяти, что улучшает затраты памяти

var sb strings.Builder

for i := 0; i < 1000; i++ {
    sb.WriteString("a")
}

Раньше для этих целей использовали bytes.Buffer

var buffer bytes.Buffer

for i := 0; i < 1000; i++ {
    buffer.WriteString("a")
}
